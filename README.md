# Emotionally Grounded Narrative Generation under AKG Constraints

This project implements a neuro-symbolic framework for controlled narrative generation using an Appraisal Knowledge Graph (AKG).

## Key Features
- Emotion trajectory planning
- AKG-constrained generation
- Emotion validation using classifier
- Retry-based control mechanism
- Traceability via Neo4j
- Ablation evaluation

## Architecture
Includes:
- Emotion Processing Layer
- Planning Layer
- Generation Layer
- Validation Layer
- Control Layer
- AKG (Neo4j)

## Evaluation Metrics
- ETVS (Emotion Trajectory Validity Score)
- Accuracy
- Soft Accuracy
- CSR (Constraint Satisfaction Rate)
- Retries

## Dataset
ROCStories subset

## Status
Research prototype (under development for publication)
